# 神经网络的基础

由于大多数的数据都是非线性的，所有分类问题有非线性的需求。利用神经网络来生成一个非线性的分类器可以比较好的实现分类功能。那么如何训练一个神经网络呢？神经网络的原理是什么？

## 神经元

一个神经元就是一个单独的计算单元，将n个输出经过计算处理之后生成一个单独的输出。通过不同的参数来区分是哪一个神经元的输出，这个参数也叫做这个神经元的权重。

对于一个神经元来说，一个经常选择的函数就是sigmoid函数。这个神经元将一个 $n$维的输入向量转换成为一个标量。公式如下：

$$
a=\frac{1}{1+exp(-(w^Tx+b))}
$$

在上述公式中， $x$为输入的 $n$维向量， $b$为偏置值， $w^T$为权重（也是这个神经元的参数）

## 单层神经元

发散思维到利用多个神经元来解决问题，也就是将输入向量喂给多个神经元，如下图所示。

所以不同神经元的参数为 $\{w^{(1)}, w^{(2)}, …, w^{(m)}\}$，偏置为$\{b_{1}, b_{2}, …, b_{m}\}$，对应的输出为 $\{a_1, a_2, ..., a_m\}$

$$
a_i=\frac{1}{1+exp(-(w^{(i)T}x+b_i))}
$$

## 反向传播

反向传播充分利用了函数微分学里面的链式法则，由于每一个参与影响函数变化的参数都可以通过链式法则来求出这个参数的梯度，那么就可以利用链式法则来简化每一个参数梯度的计算。

如何理解反向传播可以简化参数梯度的计算呢？

以一个简单的例子来说明：

有一个简单的函数：

$$
f(x,y,z)=(x+y)z
$$

不妨设 $q=(x+y)$

![[IMG-20260116173227731.png]]

对于这个函数来说，假设三个参数的初始值为 $x=-2,y=5,z=-4$

如果不利用反向传播也就是链式法则的原理，三个参数的梯度计算如下：

$$
\frac{\partial f}{\partial x}=\frac{\partial f}{\partial q}\frac{\partial q}{\partial x}\\ \frac{\partial f}{\partial y}=\frac{\partial f}{\partial q}\frac{\partial q}{\partial y}\\ \frac{\partial f}{\partial z}=\frac{\partial f}{\partial q}
$$

如果不利用链式法则所带来的便利性，那么其中 $\frac{\partial f}{\partial q}$这一个需要计算三次，大大增加了计算的负担。

接下来解释反向传播的原理：

对于上面这个计算图来说，每一步都有一个微分，但是这个微分不是指整体函数对这个参数的微分，而是这个参数所立即影响的不论是中间变量还是最终变量的微分，也就是如上图所示$\frac{\partial q}{\partial x}\frac{\partial q}{\partial y}$等，是目前这个参数对立即会影响的参数，这两个参数之间的微分。

那么得到了每个参数的这个微分之后，反向传播也就可以开始了，对于最后一步 $\frac{\partial f}{\partial f}$，很明显为1。然后对于立即影响 $f$的两个参数 $q$和 $z$而言，这两个参数的梯度就是他们之前计算的微分 $\times$ 所立即影响的那个参数的梯度。

这样子每一个参数的梯度里面所需要计算的式子都只需要计算一遍，不会重复计算，大大节约了计算力。

# 训练神经网络的一些技巧

## 正则化

对于神经网络而言，很容易出现过拟合的情况，即在训练集上性能出色，但是泛化能力一般，在除训练集以外的地方性能一般。

一般解决过拟合的问题的常用方法就是在损失函数上添加一个正则项。新的损失函数如下：

$$
J_R=J+\lambda\sum^{L}_{i=1}||W^{(i)}||_F
$$

上述公式是添加了一个L2正则项的结果，其中 $\lambda$是一个超参数，决定了正则项所影响的作用大小， $\sum^{L}_{i=1}||W^{(i)}||_F$这个是Frobenius 范数，为矩阵A各项元素的绝对值平方的总和开根。

$$
\sum^{L}_{i=1}||W^{(i)}||_F=\sqrt{\sum_{i}\sum_{j}U^2_{ij}}
$$

对于正则化来说，超参数 $\lambda$是非常重要的，需要进行微调来达到一个比较不错的效果。太大或太小都会影响神经网络，太大了，会导致每一个参数都为零，然后神经网络相当于没有学习，太小这个正则项就不起作用没办法解决过拟合的问题。

## Dropout

Dropout是一个强大的正则化工具，可以有效的解决过拟合的问题。

Dropout是在训练神经网络的过程中，以一个概率抛弃这个神经元，不计算他的前反馈以及梯度，但是在训练完成之后，重新开始使用这个神经元。所以这会导致一个问题，由于训练的过程中没有使用这个神经元，但是训练完成之后又重新启用这个神经元，会导致输出出现很大的变化，一种常见的方法是在训练时，如果某个神经元以概率 p 被保留，那么其输出值要除以 p。这样做的目的是为了保证一个神经元在训练时的预期输出和在测试时的实际输出保持大致相同的尺度，从而避免输出幅度发生剧烈变化。

## 动量更新

动量更新其实就是利用momentum在原参数的更新上面进一步优化，对于参数 $\theta$的更新公式如下：

$$
m_{t+1}=\beta m_t+(1-\beta)\nabla_{\theta _t}Loss \\ \theta_{t+1}=\theta_t-\alpha m_{t+1}
$$

其中 $\alpha$是larning rate

## 自适应优化方法

对于固定的学习率有一个弊端就是容易错过最小值，所以提出一个新的方法，学习率变化的优化方法，对于每一个参数而言，都有一个学习率，某个参数的历史更新次数越少，它的学习率就越高，对于那些不经常更新的参数，我们给予它们更大的步长，让它们更快地收敛。

$$
\theta_{t,i}=\theta_{t-1,i}-\frac{\alpha}{\sqrt{\sum^{t}_{\tau=1}g^2_{t,i}}}g_{t,i}
$$

其中， $\theta_{t,i}$是第t步，第i个参数的学习率， $\theta_{t,i}$是第t步，第i个参数的学习率， $g_{t,i}$是第t步，第i个参数的梯度， $\alpha$是全局学习率， $\sqrt{\sum^{t}{\tau=1}g^2{t,i}}$是从第1步到第t步，第i个参数所有梯度的平方和。

这个公式的核心思想是：分母部分会随着训练的进行而不断累积梯度平方。如果某个参数的梯度一直很大，那么分母就会变得很大，导致其学习率 $\frac{\alpha}{\sqrt{\sum^{t}{\tau=1}g^2{t,i}}}$变小；反之，如果某个参数的梯度一直很小，那么分母就会保持较小，其学习率就会相对较高。

简易代码实现如下：

```python
# Assume the gradient dx and parameter vector x
cache += dx**2
x += - learning_rate * dx / np.sqrt(cache + 1e-8)
```

为了解决 AdaGrad 学习率单调递减的问题，RMSProp 应运而生。RMSProp 并不是简单地累积梯度的平方，而是使用梯度的平方的指数移动平均。RMSProp 引入了一个“衰减率”（decay rate），它让最近的梯度对当前学习率的影响更大，而历史久远的梯度影响更小。这样，学习率就不会无限减小，可以更好地适应非平稳的目标。

```python
# Update rule for RMS prop
cache = decay_rate * cache + (1 - decay_rate) * dx**2
x += - learning_rate * dx / (np.sqrt(cache) + eps)
```

![[IMG-20260116173227765.png]]

Adam 可以被看作是 RMSProp 的一个变种，它不仅使用了梯度的平方的移动平均（二阶矩），还结合了梯度的移动平均（一阶矩）。

```python
# Update rule for Adam
m = beta1*m + (1-beta1)*dx
v = beta2*v + (1-beta2)*(dx**2)
x = x - learning_rate * m / (np.sqrt(v) + eps)
```
